ВАРИАНТ 03

После неправильных действий пользователя была утеряна вся информация, хранившаяся на системном диске. Резервных копий дискового накопителя не оказалось, но сохранились некоторые маршруты по файловой системе (имя дисковода не указывается), например,  “WINNT\SУSТЕМ32\CERTSRV\CERTCO~1\Х86”. 

Требуется восстановить иерархическую (древовидную) структуру каталогов в файловой системе по списку всевозможных маршрутов по файловой системе.

Входные данные

Первая строка входного файла содержит значение N (1 <= N <= 500) – количество различных маршрутов по файловой системе. Далее следует N строк с маршрутами. Каждый маршрут занимает одну единственную строку и не содержит пробелов (в том числе лидирующих и хвостовых пробелов). Длина маршрута не превышает 80 символов. Каждый маршрут встречается ровно один раз и представляет собой цепочку подчиненных (включенных друг в друга) имен каталогов, разделяемых обратной косой чертой (“\”). Имя каждого каталога состоит из заглавных букв английского алфавита, цифр и специальных символов (“!#$%&’()-@^_`{}~”). Длина имени каталога не превышает 8 символов.

Выходные данные

Выходной файл представляет собой иерархическую (древовидную) структуру каталогов в файловой системе. Имя каждого каталога должно встречаться ровно один раз и печататься на отдельной строке. Имени подчиненного каталога должен предшествовать один или несколько символов табуляции. Количество символов табуляции должно равняться уровню вложенности подкаталога. Все подчиненные каталоги одного уровня должны быть напечатаны в лексикографическом порядке следом за родительским каталогом. Подкаталоги верхнего уровня должны быть напечатаны в лексикографическом порядке, и не иметь отступов вправо, т. е. символов табуляции.

Пример входных данных (файл dtree.in)

7
WINNT\SYSTEM32\CONFIG
GAMES
WINNT\DRIVERS
HOME
WIN\SOFT
GAMES\DRIVERS
WINNT\SYSTEM32\CERTSRV\CERTCO~1\X86

Соответствующие выходные данные (файл dtree.out)	

GAMES
	DRIVERS
HOME
WIN
	SOFT
WINNT
	DRIVERS
	SYSTEM32
		CERTSRV
			CERTCO~1
				X86
		CONFIG
